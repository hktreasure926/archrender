# Studio Nomad - Branding Guidelines

## Brand Overview

**Studio Nomad** is a high-ticket tech-enabled architectural visualization and design iteration platform. The brand reflects precision, elegance, and professional architectural aesthetics.

## Design Philosophy: Neo-Bauhaus

The Studio Nomad interface follows Neo-Bauhaus design principles:
- Warm sands, charcoal, brass accents
- Structural layouts
- Minimal shadows
- 1px borders

## Component Guidelines

### Buttons
- Border radius: 8px
- Padding: 12px 24px
- Transition: 200ms ease-out
- Hover: Scale 1.02, shadow-md
- Active: Scale 0.95

### Cards
- Border radius: 12px
- Border: 1px solid #E5E2DC
- Shadow: None (minimal shadows per Neo-Bauhaus)
- Padding: 16px or 20px

### Inputs
- Border radius: 8px
- Border: 1px solid #E5E2DC
- Focus: Border color #C5A059
- Padding: 8px 12px

## Logo

**Studio Nomad** logo should appear in:
- Header/navigation
- Login page
- About/landing page

## Usage Guidelines

1. **Consistency**: Always use the defined color palette
2. **Whitespace**: Use generous spacing for clean layouts
3. **Typography**: Stick to Playfair Display for headings, Inter for body
4. **Borders**: Keep borders subtle (1px, light colors)
5. **Shadows**: Minimal to no shadows (Neo-Bauhaus principle)
6. **Images**: Use high-quality architectural renders
